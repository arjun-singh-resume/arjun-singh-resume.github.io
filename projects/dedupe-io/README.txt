
DEDUPE-IO ENTERPRISE SYSTEM
===========================
Version: 2.1.0
Release Date: 2025
Developer: Arjun Singh
OVERVIEW
________________________________________
DEDUPE-IO is a comprehensive enterprise data management system designed for
customer deduplication, high-risk profiling, and eligibility assessment.
The application features advanced risk calculation, real-time validation,
and automated reporting capabilities.
QUICK START
________________________________________
1.	EXTRACT to D:\ (Mandatory)
o	Download DEDUPE-IO.zip
o	Extract directly to D:\
o	This creates D:\DEDUPE-IO\ automatically
2.	LAUNCH Application
o	Navigate to D:\DEDUPE-IO\
o	Double-click DEDUPE-IO.accde
3.	LOGIN
o	Use credentials displayed on login screen
o	Multiple demo accounts available
o	No admin approval required for testing
SYSTEM REQUIREMENTS
________________________________________
•	Microsoft Access 2016 or later
•	Windows 10/11 Operating System
•	D: Drive with 500MB free space
•	Microsoft Excel (for export features)
•	2GB RAM minimum (4GB recommended)
KEY FEATURES
________________________________________
✓ High-Risk Customer Profiling
✓ Real-time Eligibility Validation
✓ Automated Risk Scorecards
✓ Customer Deduplication
✓ Bulk Data Processing
✓ Excel Export & Reporting
✓ CAF Tracking System
✓ User Management
